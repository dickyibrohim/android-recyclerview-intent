# Android RecyclerView Intent

Example project demonstrating RecyclerView with intent navigation to another Activity in Android Studio.

## Scope
- RecyclerView adapter pattern
- Intent-based navigation
- Simple, clear structure

## Usage
- Open in Android Studio
- Run on emulator or device
